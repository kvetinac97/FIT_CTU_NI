= Tutorial 1

== Virtualenv

[source,bash]
----
# https://virtualenv.pypa.io/en/stable/userguide/

# create env
virtualenv ddw-tutorial
# virtualenv --system-site-packages ddw-tutorial

# activate
source ddw-tutorial/bin/activate

# operations ...
pip install ...

# deactivate and remove
deactivate
rm -r ./ddw-tutorial
----

== Requirements

[source,bash]
----
pip install -r requirements.txt
----
